# Equinox Kernel for Poco F5 Pro (mondrian)

## Features
- **Core:** Android Common Kernel (GKI) 5.10
- **Root:** Integrated KernelSU-Next
- **Security:** SUSFS support (Kernel level)
- **Networking:** WireGuard support
- **Compiler:** Neutron Clang (O3 Optimized)
- **Developer:** LF5

## Build Information
Built using GitHub Actions with LTO and ZSTD compression.
